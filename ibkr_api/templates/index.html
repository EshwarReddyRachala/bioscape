Welcome to the IBKR API

This is a Flask-based API for trading with Interactive Brokers.

Available Endpoints

  • Check API Status - Verify if the API is running.
  • Place an Order - Submit a trade order.
  • Get Market Data - Retrieve stock price data.
  • Account Information - View trading account details.

About the API

This API is built using Python and Flask and integrates with Interactive Brokers' TWS API to provide seamless trading automation. Features include:

  • Real-time market data access
  • Order placement and execution
  • Portfolio management
  • Historical data retrieval

Getting Started

To start using the API, ensure you have Interactive Brokers' Trader Workstation (TWS) running and API access enabled.
